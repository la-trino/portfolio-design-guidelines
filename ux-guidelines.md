# Personal Portfolio - UX Guidelines

## Sumary
Esta documentación resume los requerimientos funcionales y de contenido para el diseño y posterior implementación de un portafolio web personal para una diseñadora gráfica e ilustradora, a publicarse con GitHub Pages. La lógica del diseño está basada en el libro "The Elements of User Experience" de Jesse James Garret. 

## 1.1 Objetivos del sitio
- El propósito del sitio es servir como Portafolio web de proyectos de diseño gráfico e ilustración. El sitio debe constar de una página de inicio con un menú de proyectos, y las páginas de los proyectos en sí, compuestas cada una de texto, imagen y video.
- El sitio debe ser fácilmente actualizable, es decir, se deben poder subir nuevas entradas de proyectos y casos de estudio en las carpetas del sitio y a su vez deben visualizarse correctamente en el buscador web una vez hecha la actualización. 
- El sitio debe ser capaz de despplegar rápidamente tanto video como imagen.

## 1.2 Necesidades de usuario
- El sitio web está orientado a clientes del diseñador dueño del portafolio.
