# Personal Portfolio - UX Guidelines

## Sumary
Esta documentación resume los requerimientos funcionales y de contenido para el diseño y posterior implementación de un portafolio web personal, a publicarse con GitHub Pages. La lógica del diseño está basada en el libro "The Elements of User Experience" de Jesse James Garret. 

## 1.1 Objetivos del sitio
- El propósito del sitio es servir como Portafolio web de proyectos de diseño gráfico e ilustración. 
Fácilmente actualizable, despliegue de imagen y texto en un mismo espacio.
Capacidad de desplegar rápidamente tanto audio como video
Navegación flexible, por medio (video, imagen, animación) e industria (Audiovisual, Digital - RRSS, Editorial,  Interactivo)
Con casos de estudio, blog de reflexiones, about me con cv
